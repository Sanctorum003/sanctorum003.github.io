    KMP

    王道方法

    1) next 数组从1开始

    2) next[1] = 0
    next[2] = 1
    next[0]不使用

    3) 求当next数组当前位置j

    // 主要方法就是 将前一个位置(j-1)的字符与next[j-1]所对应的字符比较相同则 next[j] = next[j-1]+1;
// 不同则 将next[j-1]的字符 与next[ next[j-1] ]对应否字符相比较相同则 next[j] = next[ next[j-1] ]+1;
// 不断重复
// 如果next[...] = 0 ==> next[j] = 1;
    k = next[j-1]; 
    while(1)
    {
        if(S[k] == S[j-1])//S为字符数组
        {
            next[j] = k+1;
            break;
        }
        else
            k = next[k];
    }
